# 📊 HR Data Analytics Dashboard

## 🧾 Overview

This project presents an **HR Data Analytics Dashboard** built to analyze workforce data and provide actionable insights into employee distribution, salary trends, demographics, and organizational growth.

The dashboard helps HR teams and decision-makers monitor key metrics and make data-driven decisions to improve workforce efficiency and planning.

---

## 🎯 Objectives

* Analyze employee headcount and distribution across roles
* Understand salary trends based on qualifications
* Evaluate workforce demographics (age, gender)
* Track employee growth over time
* Identify patterns in leave behavior

---

## 📌 Key Metrics

* **Total Employees:** 161
* **Average Salary:** 54.23K
* **Average Leaves:** 16.42
* **Employees with Leave Balance > 20 Days:** 29

---

## 📊 Dashboard Insights

### 1. Employee Distribution by Role

* Shows how employees are spread across different job roles
* Helps identify workforce concentration and gaps

### 2. Gender Distribution

* 55% Female workforce
* Helps track diversity and inclusion

### 3. Age Distribution

* Visual representation of employee age groups
* Useful for workforce planning and succession strategy

### 4. Salary vs Qualification

* Compares salaries across education levels:

  * Bachelor’s Degree
  * Diploma
  * High School
  * Master’s Degree
* Helps understand compensation trends

### 5. Workforce Growth Trend

* Tracks hiring trends over time
* Shows cumulative headcount increase

## 🛠️ Tools & Technologies

* **Power BI** – Dashboard creation & visualization
* **SQL** – Data extraction and querying
* **Python (Pandas, Matplotlib/Seaborn)** – Data cleaning & EDA

---

## 🔍 Data Analysis Process

1. **Data Collection** – HR dataset (employee details, salary, roles, etc.)
2. **Data Cleaning** – Handling null values, formatting
3. **Exploratory Data Analysis (EDA)** – Identifying patterns & trends
4. **Data Modeling** – Structuring data for dashboard
5. **Visualization** – Building interactive dashboard in Power BI
